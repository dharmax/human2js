# @dharmax/human2js

High-fidelity English-to-JavaScript compilation and persistent orchestration engine.

## Features
- **Compiler**: Turns natural language into executable, high-fidelity JavaScript plans.
- **Orchestrator**: Persistent, state-based execution environment for LLM-generated JavaScript.
- **Sandbox**: Secure execution via Node.js `vm` with service injection.
